# myFlix Angular Client

A responsive Angular (standalone components) client for the myFlix movie application. Users can register, log in, browse movies, open dialogs for movie details (genre/director/synopsis), manage a simple favorites UI, and view/edit their profile.

## Live Demo (GitHub Pages)
https://dpatil1719.github.io/myFlix-Angular-client/

## Repository
https://github.com/dpatil1719/myFlix-Angular-client

## Features
- Welcome view with Sign Up and Login dialogs (Angular Material)
- Movies view (grid of movie cards)
- Dialogs for:
  - Genre
  - Director
  - Synopsis
- Favorites UI toggle (icon fill/unfill)
- User Profile view:
  - view/edit user fields
  - logout
- Routing:
  - `/welcome`
  - `/movies`
  - `/profile`

## Tech Stack
- Angular (standalone setup)
- TypeScript
- Angular Material
- RxJS
- GitHub Pages (deployment)
